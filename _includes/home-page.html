Used Technologies

EOS

EOS is a decentralized operating system based on blockchain technology. It is the basis for cubebased.

ipfs.tech

The InterPlanetary File System (IPFS) is a protocol and peer-to-peer network for storing and sharing data in a distributed file system. IPFS uses content-addressing to uniquely identify each file in a global namespace connecting all computing devices.

web3.storage

Web3.Storage is a service to make building on top of Filecoin. Content uploaded to Web3.Storage is pinned redundantly three times to an IPFS Cluster of 18+ geographically distributed nodes.

Flutter

Is an open-source UI SDK created by Google. It is used to develop cross-platform applications for Android, iOS, Linux, macOS, Windows, Google Fuchsia, and the web from a single codebase.
